ctranslate2.converters.OpenNMTTFConverter

class ctranslate2.converters.OpenNMTTFConverter

Converts models generated by OpenNMT-tf.

Inherits from: ctranslate2.converters.Converter

__init__(model_spec: ModelSpec, src_vocab: Union[str, List[str]], tgt_vocab: Union[str, List[str]], model_path: Optional[str] = None, variables: Optional[Dict[str, ndarray]] = None)

Initializes the OpenNMT-tf converter.

Parameters

  • model_spec – Specification of the model to convert.

  • src_vocab – Path to the source vocabulary or list of tokens.

  • tgt_vocab – Path to the target vocabulary or list of tokens.

  • model_path – Path to the OpenNMT-tf checkpoint (mutually exclusive with variables).

  • variables – Dictionary of variables name to value (mutually exclusive with model_path).

Raises

ValueError – If none or both of model_path and variables are set.

convert(output_dir: str, vmap: Optional[str] = None, quantization: Optional[str] = None, force: bool = False)

Converts the model to the CTranslate2 format.

Parameters

  • output_dir – Output directory where the CTranslate2 model is saved.

  • vmap – Optional path to a vocabulary mapping file that will be included in the converted model directory.

  • quantization – Weight quantization scheme (possible values are: int8, int8_float16, int16, float16).

  • force – Override the output directory if it already exists.

Returns

Path to the output directory.

Raises

  • RuntimeError – If the output directory already exists and force is not set.

  • NotImplementedError – If the converted cannot convert this model to the CTranslate2 format.
